# Select Queries In My SQL

## Select The Entire Table In MySQL

```sql
select * from studentdemodb;
```

## Select Multiple Columns From a Table in MySQL

``` sql
select empnam, empprofile from studentdemodb;
```

## Select Single Column From Table In MYSQL

``` sql
select empnam from studentdemodb;
```

## Where Clause In MYSQL

``` sql
select * from studentdemodb where empnam = "Akash Patel";
```
